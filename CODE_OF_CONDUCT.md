# Code of Conduct

## Our pledge

We want `md-redline` to be a welcoming, low-friction project for contributors of all experience levels. We expect everyone participating in the project to be respectful, constructive, and generous with context.

## Expected behavior

- Be respectful and assume good intent.
- Give actionable technical feedback without insults or personal attacks.
- Help make the project accessible to newcomers.
- Respect differing viewpoints and lived experiences.

## Unacceptable behavior

- Harassment, intimidation, or discriminatory language
- Personal attacks or sustained hostility
- Publishing private information without permission
- Any conduct that would make collaboration feel unsafe

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, and other contributions that violate this Code of Conduct.

## Reporting

For conduct concerns, please contact the maintainer privately rather than starting a public thread.
